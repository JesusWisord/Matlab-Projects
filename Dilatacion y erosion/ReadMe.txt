Nuestros programas requieren la presencia de la imagen
"OutputBaselineBrain.png" en la misma carpeta de los programas.
Cada programa guardará imágenes mostrando los resultados
de los mismos programas. Las imagenes tendran un nombre
explícito acerca de su origen.

Apertura:

Al ejecutarse guardará dos imágenes, una apertura utilizando una linea de 1x3 y
y un cuadrado de 3x3. Ambos se guardarán como "Apertura Linea.jpg" y "Apertura
Rectangulo.jpg".

Cerradura

Al ejecutarse guardará dos imágenes, una apertura utilizando una linea de 1x3 y
y un cuadrado de 3x3. Ambos se guardarán como "Cerradura Linea.jpg" y "Cerradura
Rectangulo.jpg".

Dilatación

Aquí utilizamos los elementos estructurales de Linea y Cruz con matriz Manual
y utilizando el comando de Strel. Las imagenes guardadas explicaran que se
utilizo en cada caso. "Dilatacion Comando" se refiere a que se usó comando para
dilatar la imagen. "Dilatacion Manual" se utilizaron ciclos para dilatarla.
"strel linea" quiere decir que se usó el comando strel para crear la
imagen estructural de una linea. "matriz linea" quiere decir que se declaró
como matriz la imagen estructural para la dilatacion.
Entonces "Dilatacion Comando Strel Linea.jpg" quiere decir que la dilatacion se
efectuó por medio de comando con strel de linea como elemento estructural.

 Las imagenes guardadas explicaran que se utilizo en cada caso. "Erosion Comando"
