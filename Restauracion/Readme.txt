Tratamiento de Imágenes Médicas
Práctica #7

Duarte García, Jesús.
Nava Vargas, Cathia Michelle.

El Programa consta de 4 programas. Se recomienda copiar el
programa a una carpeta independiente con la imagen "rm_rodilla.jpg"
dentro de cada carpeta para apreciar los resultados con mayor
facilidad.

EtapaI

Este programa mostrará una ventana con 3 imágenes y su respectivo
título. Deberá presionarse una tecla para poder ver el siguiente
filtro, en total mostrará 4 filtros diferentes los mismos que
seran guardados en la carpeta que contiene el programa con un
nombre que expique que filtro es. Igualmente la imagen corrompida
se encontrará en el programa.

EtapaII

Este programa mostrará 6 imágenes divididas en 2 ventanas.
Cada una tendrá un título explicando que es cada una.
Cada ventana mostrará la imagen original o la imagen corrompida
y el filtro aplicado a su respectiva imagen. Las imagenes
filtradas serán guardadas en la carpeta donde se encuentre el programa.

EtapaIII

El programa mostrará dos imágenes con la detección de bordes de la
imagen original, una manual y otra mediante un comando. Estas dos imágenes
se guardaran en la misma carpeta del programa con un nombre que indique
cual es cada una.

EtapaIV

Este programa mostrará dos ventanas con una imagen, las imagenes
mostradas son la imagen original o corrompida y el filtro aplicado.
Cuando se muestre cada imagen se pausará el programa para poder
apreciar las diferencias y continuara al presionar cualquier tecla.
Todos los resultados se guardaran en la misma carpeta donde se
encuentre el programa con un nombre explicito sobre lo que representa.
